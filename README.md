# JavaScript Basics Practice

## About This Repository

This repository contains the JavaScript programs I practiced while learning the fundamentals of JavaScript. These exercises were given to help me understand the basics of programming and improve my logical thinking.

Each program focuses on a specific concept such as functions, loops, conditions, mathematical operations, and pattern printing. Solving these problems helped me understand how to write clean and structured JavaScript code.

## Concepts Covered

- Functions
- Arrow Functions
- Variables and Data Types
- Conditional Statements
- Loops (`for` and `while`)
- Nested Loops
- Mathematical Operations
- Problem Solving

## Programs

1. Even or Odd Checker
2. Pass or Fail Result Evaluator
3. Largest of Three Numbers
4. Sum from 1 to N
5. Multiplication Table Generator
6. Digit Counter
7. Number Reverser
8. Factorial Calculator
9. Prime Number Checker
10. Star Pattern Builder

## What I Learned

- Writing reusable functions
- Using arrow functions
- Working with `if...else` conditions
- Using `for` and `while` loops
- Solving number-based problems using mathematical operations
- Building simple logic without using built-in shortcuts
- Improving problem-solving skills through practice
